In this section there is a topic about **Block level element** and and **inline level element**.
Element means a _tag_ which has content in it is know as element.

### Block level element:

An element which occupies all the area in which line in is written (from left to right of the screen**).
**Example:\*\*

- `<p>`(paragraph tag).
- `<h1>`upto `<h6>` (heading tags) etc.

### Inline level element:

An element which occupies just its text area not the whole line.
**Example:**

- `<img>` (image tag).
- `<a href>` (link tag).
